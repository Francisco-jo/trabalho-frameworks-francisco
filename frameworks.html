Tutorial: Frameworks Front-End

Frameworks como Bootstrap e jQuery são amplamente utilizados para agilizar o desenvolvimento web.

Adicionar vários novos elementos com after() e before()

O que cada botão faz:
  • Texto a frente: adiciona um novo texto antes do elemento principal.
  • Texto atras: adiciona um novo texto depois do elemento principal.
  • Remover: apaga todos os textos adicionados pelos dois botões acima.

        $(document).ready(function(){
          $("#btn1").click(function(){
            $("img").before('o texto comerça aqui,');
          });
          $('#btn2').click(function () {
              $('img').after(' , o texto termina aqui');
            });
          $('#remover').click(function () {
              $('.gerado1').remove();
              $('.gerado2').remove();
            });
        });


Bootstrap: Alerta (Alert)

O componente alert exibe mensagens de sucesso, erro, aviso ou informação de forma destacada.

<div class="alert alert-success">Cadastro realizado com sucesso!</div>
Cadastro realizado com sucesso!

Bootstrap: Accordion

O accordion permite esconder e mostrar conteúdo de forma organizada.

<div class="accordion">...</div>

É a parte visual de um site, onde o usuário interage.

É um framework CSS que facilita a criação de interfaces responsivas.

jQuery: Mostrar/Esconder com slideToggle()

Com o jQuery podemos esconder ou mostrar conteúdos com animação.

$("#btnToggle").click(function() {
   $("#texto").slideToggle();
   });

Essa explicação aparece com jQuery usando slideToggle()!

jQuery: Efeito Hover

Podemos aplicar efeitos ao passar o mouse em um elemento com jQuery.

$(".box").hover(function() {
    $(this).css("background-color", "#d4edda");
    }, function() {
    $(this).css("background-color", "#f8f9fa");
   });
Passe o mouse aqui para ver o efeito com jQuery.
