Curso de Sistemas para Internet

O curso tem como objetivo formar profissionais aptos a desenvolver soluções tecnológicas para a web, com conhecimentos sólidos em programação, banco de dados, usabilidade e infraestrutura.

Objetivos do Curso

  • Formar profissionais com visão crítica, empreendedora e inovadora.
  • Capacitar no desenvolvimento de aplicações web responsivas e seguras.
  • Promover a compreensão de tecnologias front-end e back-end.

Grade Curricular (exemplo)

  • Lógica de Programação
  • Introdução à Web
  • Fundamentos de Informática
  • Matemática Aplicada

  • Programação Orientada a Objetos
  • Banco de Dados
  • Design para Web
  • Redes de Computadores

Áreas de Atuação

Desenvolvedor Web

Criação de sites, sistemas e aplicativos web front-end e back-end.

Administrador de Sistemas

Gerência de servidores, redes e infraestrutura de TI.

Analista de Dados

Trabalho com análise, manipulação e visualização de dados web.
